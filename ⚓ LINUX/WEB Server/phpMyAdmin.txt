*** Установка phpMyAdmin
--- $ sudo apt install phpmyadmin php-mbstring php-zip php-gd php-json php-curl

* Если вы следовали предварительному руководству по стеку LAMP некоторые из этих модулей будут установлены вместе с php пакетом

* Рекомендуется также установить следующие пакеты
--- $ sudo apt install phpmyadmin php-mbstring php-zip php-gd php-json php-curl

php-mbstring   модуль для управления строками и преобразования строк в другие кодировки
php-zip        расширение поддерживает загрузку .zip файлов в phpMyAdmin
php-gd         включает поддержку графической библиотеки GD
php-json       обеспечивает PHP поддержкой реализации JSON
php-curl       позволяет PHP взаимодействовать с различными типами серверов

* Вот параметры, которые вы должны выбрать при появлении запроса, чтобы правильно настроить установку

Для выбора сервера выберите apache2
Предупреждение. При появлении приглашения «apache2» выделяется, но не выбран. Если вы не нажмете SPACEвыбрать Apache, программа установки не переместит необходимые файлы во время установки. Нажмите SPACE, TAB, а затем ENTERвыберите Apache.

Выберите Yes, когда вас спросят, следует ли использовать dbconfig-commonдля настройки базы данных
Затем вам будет предложено выбрать и подтвердить пароль приложения MySQL для phpMyAdmin.
